## Getting Started

### Creating Your Account

1. **Visit the platform** at app.thequantumclub.com
2. **Choose sign-up method**:
   - Email & Password
   - Google OAuth
   - Apple Sign-in
   - LinkedIn OAuth

3. **Complete email verification** (if using email signup)
   - Check your inbox for verification link
   - Click link to activate account
   - Return to platform and log in

4. **Accept invite code** (if invited by strategist)
   - Enter invite code during signup
   - Your profile will be auto-linked to strategist

### First Login

After your first login, you'll be guided through:
1. **Profile basics**: Name, title, location
2. **Work preferences**: Remote/hybrid/onsite, industries
3. **Privacy settings**: Control who sees what
4. **Calendar connection**: Sync availability for interviews

## Profile Setup

### Essential Information

**Complete these sections for best results:**

1. **Professional Summary**
   - 2-3 sentences about your expertise
   - Key achievements or unique value
   - Career goals

2. **Work Experience**
   - Current & past roles
   - Key accomplishments (not just duties)
   - Technologies used

3. **Skills**
   - Technical skills (programming languages, tools)
   - Soft skills (leadership, communication)
   - Certifications

4. **Education**
   - Degrees earned
   - Relevant coursework
   - Academic honors

### Upload Your Resume

1. Navigate to **Profile → Documents**
2. Click **Upload Resume**
3. Select PDF or DOCX file (max 5MB)
4. Resume will be auto-parsed to fill profile
5. Review and adjust parsed data

### Profile Photo

1. Go to **Profile → Edit Profile**
2. Click on avatar circle
3. Upload image (JPG, PNG, max 5MB)
4. Crop to square format
5. Save changes

## Security & Privacy

### Setting Up Multi-Factor Authentication (MFA)

**Highly recommended for account security**

1. Navigate to **Settings → Security**
2. Click **Enable MFA**
3. Scan QR code with authenticator app:
   - Google Authenticator
   - Authy
   - Microsoft Authenticator
4. Enter 6-digit code to verify
5. **Download recovery codes** (store securely)
6. MFA now required at login

### Privacy Controls

**Control what information companies can see:**

1. **Profile Visibility**
   - Public: Anyone can find you
   - Private: Only matched companies see profile
   - Stealth: Completely hidden

2. **Field-Level Visibility**
   - Hide current salary
   - Hide phone number
   - Hide email address
   - Show only after mutual match

3. **Employer Shield**
   - Block current employer from seeing profile
   - Add up to 5 domain blocklists
   - Example: block `@currentcompany.com`

**To add blocked domain:**
1. Go to **Settings → Privacy**
2. Scroll to **Employer Shield**
3. Enter company domain (e.g., `acme.com`)
4. Click **Add Domain**

### Data Export & Deletion

**GDPR Compliance: Your data, your control**

**Export all data:**
1. Settings → Privacy → Export Data
2. Receive JSON file via email
3. File includes: profile, applications, messages, settings

**Delete account:**
1. Settings → Account → Delete Account
2. Enter password to confirm
3. 30-day soft delete (can restore)
4. After 30 days: permanent deletion

## Job Applications

### Finding Jobs

1. **Browse Jobs page**
   - Filter by: role, level, location, remote, comp
   - Save interesting roles

2. **Get Matched**
   - QUIN AI recommends roles based on profile
   - See match score (0-100%)
   - Review "Why matched" reasoning

### Applying to Jobs

**Two application methods:**

**Method 1: Manual Apply**
1. Click job card
2. Review job description
3. Click **Apply Now**
4. Customize cover letter (optional)
5. Submit application

**Method 2: Club Sync (Auto-Apply)**
1. Enable in **Settings → Preferences**
2. QUIN auto-applies to best matches
3. Review submitted applications daily
4. Opt-in required for each role type

### Application Pipeline

**Your application moves through stages:**

1. **Applied** → Resume reviewed
2. **Screen** → Initial call with strategist
3. **Interview** → Company interviews (1-3 rounds)
4. **Final** → Final decision stage
5. **Offer** → Offer received
6. **Hired** → Accepted offer 🎉

**Track progress:**
- Dashboard shows all active applications
- Receive email updates at each stage
- SLA timers show expected next action

### Interview Preparation

**Tools provided:**

1. **Smart Questions**
   - QUIN generates 7 questions to ask interviewer
   - Based on job description + company research

2. **Panel Briefing**
   - Background on interviewers (from LinkedIn)
   - Communication styles
   - What they value

3. **Practice Session**
   - Mock interview with QUIN
   - Voice-based Q&A
   - Feedback on answers

## Troubleshooting

### Can't Log In

**Issue**: "Invalid email or password"

**Solutions:**
1. Check email spelling
2. Try password reset
3. Check if MFA is enabled (need code)
4. Clear browser cache
5. Try incognito/private mode

### Profile Photo Won't Upload

**Issue**: Upload fails or image distorted

**Solutions:**
1. Check file size (max 5MB)
2. Use JPG or PNG format
3. Ensure stable internet connection
4. Try different browser
5. Compress image if too large

### Not Receiving Emails

**Issue**: Verification/reset emails not arriving

**Solutions:**
1. Check spam/junk folder
2. Add `@thequantumclub.com` to contacts
3. Check email spelling in profile
4. Wait 5 minutes (emails can be delayed)
5. Request new email (limit: 5/hour)

### Job Matches Seem Off

**Issue**: QUIN recommending irrelevant roles

**Solutions:**
1. Update profile with more detail
2. Refine preferences (Settings → Preferences)
3. Mark matches as "Not interested" to train AI
4. Ensure skills section is complete
5. Check compensation range is realistic

### Application Stuck in Stage

**Issue**: No updates in 2+ weeks

**Actions:**
1. Check SLA timer on application card
2. Contact your strategist directly
3. Dashboard shows expected next action
4. Escalate if SLA exceeded
5. Email: support@thequantumclub.com

### Privacy Concern

**Issue**: Current employer might see profile

**Immediate actions:**
1. Add domain to **Employer Shield** (Settings → Privacy)
2. Switch profile to **Private** visibility
3. Disable Club Sync temporarily
4. Contact strategist to review active applications
5. Report concern: privacy@thequantumclub.com
