# Internationalization (ClubOS)

**Mindset:** see **`AGENTS.md`** — lead the user, parallelize when possible, ship excellence in the ask (not only minimal patches).

All agents must internationalize **user-visible** UI copy in `src/**/*.tsx`.

- Use `react-i18next` (`useTranslation`, `t()`, or `Trans`).
- Store English strings in `src/i18n/locales/en/<namespace>.json`; add other locales when the task includes them.
- Do not use raw JSX text for prose (ESLint: `i18n/no-raw-jsx-text`).
- Exceptions: `src/components/ui/**`, tests, `src/pages/legal/**`, non-UI technical strings.

**Full agent playbook** (scope, Supabase, tests, anti-patterns): **`AGENTS.md`** at the `ClubOS` root.
